# Redux

## Como utilizar o Redux?

#### Store

- Configuração global do Redux;
- Estado global do Redux;

#### Action

- Função/método que dispara algo, ou seja, uma ação mesmo;

#### Reducer

- Configurações de estados especificos/menores
